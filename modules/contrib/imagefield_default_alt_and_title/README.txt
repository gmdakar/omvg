INTRODUCTION
------------

The module "Imagefield default alt & title" adds alts and titles for images
to where they are not, taking them out of the title material.

REQUIREMENTS
------------

This module requires the following modules:

 * Entity API (https://www.drupal.org/project/entity)

INSTALLATION
------------

0 - Prerequisites:
Requires the entity module to be enabled.

1 - Run composer require 'composer require drupal/modulename'
From the modules administration/management page (requires the entity module
to be enabled).
More information at:
https://www.drupal.org/docs/8/extending-drupal-8/installing-drupal-8-modules


CONFIGURATION
-------------

* You may want to disable Toolbar module, since its output clashes with
   Administration Menu.
